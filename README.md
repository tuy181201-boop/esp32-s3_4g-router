Этот проект превращает ESP32-S3 и модем A7670E в полноценный переносной 4G роутер. Устройство подключается к сотовой сети (LTE), получает IP через PPP, раздаёт интернет по Wi-Fi (точка доступа) и предоставляет веб-страницу со статусом. Управление и телеметрия — через MQTT. Состояние отображается на семисегментном индикаторе.

Основные возможности (списком)

· ✅ Поддержка модема A7670E (A7672, A7670C, SIM7600 можно адаптировать)
· ✅ PPP-соединение с автоматическим переподключением
· ✅ NAT (Network Address Translation) — устройства в локальной сети выходят в интернет через 4G
· ✅ Встроенная Wi-Fi точка доступа (AP), до 4 клиентов
· ✅ Веб-сервер со статусом: число клиентов, уровень сигнала, оператор, uptime
· ✅ MQTT-телеметрия (клиенты, RSSI, статус) и команды (перезагрузка)
· ✅ Семисегментный дисплей (через 74HC595): показывает количество подключённых клиентов
· ✅ Легко конфигурируется через menuconfig или файл настроек

Подключение (таблица)

Компонент Пин ESP32-S3
UART TX (модем) GPIO 17
UART RX (модем) GPIO 18
PWRKEY GPIO 14
DTR (опционально) GPIO 15
74HC595 DS GPIO 16
74HC595 SHCP GPIO 17
74HC595 STCP GPIO 18

Примечание: используемые пины можно изменить в коде.

Настройка и сборка

```bash
git clone https://github.com/yourname/esp32-a7670e-4g-router.git
cd esp32-a7670e-4g-router
idf.py set-target esp32s3
idf.py menuconfig   # настройте Wi-Fi AP, APN, MQTT брокер
idf.py build flash monitor
```

Конфигурация перед запуском

· APN – обычно internet.tele2.ru (или вашего оператора)
· Wi-Fi SSID / пароль – измените с дефолтных
· MQTT брокер – можно оставить broker.emqx.io или поднять свой
